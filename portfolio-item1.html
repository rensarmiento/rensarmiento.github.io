Discord Streamer Bot

A bot created for streamers.

Click here for repository

I wanted to create a bot for my friends who stream on Twitch. Using Discord's Python API, I created a bot that would read the activities of the users on the server. If the user was streaming on Twitch, the other users would be able to see the streamers on the right -hand side of the screen.

The inspiration came from many of my friends who wanted to start streaming at the beginning of Covid-19. There are bots available, however a fee was required for multiple streamers on a server, thus I wanted to create a subtle way to notify others without posting a message to the server.
